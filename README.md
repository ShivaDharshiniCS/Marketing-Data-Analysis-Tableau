## ğŸ“Š KPIs Calculated
- **CTR (Click-Through Rate)** = Clicks / Impressions Ã— 100  
- **CPM (Cost per 1000 Impressions)** = (Media Cost / Impressions) Ã— 1000  
- **CPC (Cost per Click)** = Media Cost / Clicks  
- **CPA (Cost per Acquisition)** = Media Cost / Conversions  
- **Bounce Rate** = (Bounces / Visits) Ã— 100  
- **Pages per Visit** = Page Views / Visits  
- **Avg Time on Page** = Time on Site Total / Page Views  
- **Exit Rate** = (Page Exits / Page Views) Ã— 100  
- **Video Completion Rate** = Video Fully Played / Video Views Ã— 100
